# <Prework Study Guide>

## Description

Contents contain a brief overview of things learned in the prework for the start of UTA bootcamp. There were challenging lessons and obsticles to overcome. 

The main motivation for the work completed is to get a headstart and be prepared for the start of bootcamp. The building of these projects help lay some foundations to better understand what is ahead. In these lessons, we learned a few fundimentals and some best practices but know this just scratches the surface.
